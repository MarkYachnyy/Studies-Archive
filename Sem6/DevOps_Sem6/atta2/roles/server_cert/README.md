# Роль: server_cert

Роль для генерации и подписи **SSL/TLS-сертификата сервера** с использованием вашего корневого центра сертификации (Root CA).

## Описание

Роль выполняет следующие задачи:
1. Создает директорию для хранения файлов сертификата.
2. Копирует конфигурационный файл для генерации CSR (Certificate Signing Request).
3. Генерирует приватный ключ сервера и CSR.
4. Подписывает CSR корневым CA, создавая итоговый сертификат.
