# Роль: root_ca

Роль для создания **корневого центра сертификации (Root CA)** с помощью Ansible. Генерирует приватный ключ, запрос на подпись сертификата (CSR) и самоподписанный сертификат.

## Описание

Роль выполняет следующие задачи:
1. Создает директорию для хранения файлов CA.
2. Генерирует RSA-приватный ключ для Root CA.
3. Создает CSR (Certificate Signing Request) с указанными данными организации.
4. Генерирует самоподписанный сертификат Root CA.
