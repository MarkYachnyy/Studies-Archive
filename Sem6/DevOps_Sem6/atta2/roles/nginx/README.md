# Role: nginx

Роль для установки и настройки NGINX с поддержкой TLS/SSL для домена `task-service.local`.

## Описание

Роль выполняет следующие задачи:
1. Устанавливает NGINX из пакетов
2. Настраивает конфигурацию сайта с редиректом HTTP → HTTPS
3. Создает директорию для SSL-сертификатов
4. Копирует SSL-сертификат и приватный ключ
5. Проверяет конфигурацию и перезагружает NGINX
